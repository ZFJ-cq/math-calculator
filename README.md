# 全学段数学计算器

一款支持全学段数学运算的多功能计算器，包括基本运算、代数运算、几何计算、三角函数等。

## 项目描述

一款支持全学段数学运算的多功能计算器，包括基本运算、代数运算、几何计算、三角函数等。

## 功能特性

- 基本运算 - 加减乘除、平方、开方
- 代数运算 - 方程求解、多项式运算
- 几何计算 - 面积、体积计算
- 三角函数 - sin、cos、tan等
- 历史记录 - 保存计算历史

## 技术栈

- HTML5
- CSS3
- JavaScript
- Math.js

## 快速开始

直接打开 index.html 文件即可使用。
